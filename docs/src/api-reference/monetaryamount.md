## MonetaryAmount



| Field | Type | Description |
| ----- | ---- | ------------|
| value | string<br />_**required**_ | An amount of money (the fractional part is optional) |
| currency | string<br />_**required**_ | ISO 4217 3 character currency code.<br><br>Note: Lune does not support all currency codes.<br> |

##### Example
```json
{
    "value": "3.14",
    "currency": "string"
}
```
