## PaginatedBase



| Field | Type | Description |
| ----- | ---- | ------------|
| has_more | boolean<br />_**required**_ | Whether or not there are more elements available after this set. If false, this set comprises the end of the array. |

##### Example
```json
{
    "has_more": "boolean"
}
```
