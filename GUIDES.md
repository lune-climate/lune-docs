# How to write a guide?

## Paragraphs

* Add a quick summary describing the problem the guide solves.
* Add a paragraph listing all prerequisites needed to perform all the actions in the guide
* Provide an brief overview where you list ALL the steps required to implement the solution.
* Explain each step in detail

## Rules

* Every time an endpoint is referenced, provide a link to it in the API reference.
* Every time an object is referenced, provide a link to it in the API docs.
* If something is worth highlighting, is tricky or ambiguous, use a container: https://v1.vuepress.vuejs.org/guide/markdown.html#custom-containers
* Add images to provide context where you can
* Add code sample where needed. Code sample should be copy/pastable and work, except for secrets.
